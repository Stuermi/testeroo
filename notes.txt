Git Initialisieren:
    git init 

Status abfragen:
    git status 

Dateien in Staging hinzufügen:
    git add "Dateiname"

Dateien in Git hochladen:
    git commit -m "Nachricht zum Änderungsstatus"

Git mit VS Code nutzen:
    über Source Control
    + um Datei in Staging hinzuzufügen
    mit blauen Button commiten
    Nachricht zum Änderungsstatus hinzufügen

Code zu Github schieben:
    git remote add origin "Link zur Github Repository" 
    git push -u origin master

Git reset Befehle:
    git reset --soft HEAD~1
        Nimmt den letzten commit zurück Änderung bleibt aber im staged repository sowie working directory
    git reset --mixed HEAD~1
        Nimmt den letzten commit zurück Änderung bleibt nur in der Working directory
    git reset --hard HEAD~1
        Löscht den letzten commit und löscht die letzte Änderung komplett
