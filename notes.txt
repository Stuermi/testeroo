Git Initialisieren:
    git init 

Status abfragen:
    git status 

Dateien in Staging hinzufügen:
    git add "Dateiname"

Dateien in Git hochladen:
    git commit -m "Nachricht zum Änderungsstatus"

Git mit VS Code nutzen:
    über Source Control
    + um Datei in Staging hinzuzufügen
    mit blauen Button commiten
    Nachricht zum Änderungsstatus hinzufügen

Code zu Github schieben:
    git remote add origin "Link zur Github Repository" 
    git push -u origin master
